Italgas Reti runs a highly-regulated priority service request process — dispatch for suspected methane leaks — with strict response-time SLAs enforced by ARERA. Mining this end-to-end (call intake → work order → field arrival → containment → close) will show which regions and shifts miss regulatory clocks and why.
