Top 3 Process Mining Use Cases

1. e-Marelli B2B Portal — Distributor Order-to-Cash

The e-Marelli B2B portal is Marelli Aftermarket's primary channel to independent workshops, distributors and the Checkstar network across 60+ countries — and two active opps are upgrading it ($178k e-Marelli upgrade at stage 03 + $85k Phase 2). Portal-order variants that fall back to phone/email inbound bleed the 278-seat Service Cloud call-center budget and slow cash conversion. AFPI mines the end-to-end distributor-order event log (B2B Commerce → Sales Cloud → SAP OTC → shipment) to quantify self-service leakage, exception rework and DSO — feeding Capgemini's upgrade with the real "as-is" map before they rebuild it. Metric moved: portal self-service rate, DSO, cost per B2B order.

Key System(s) to be Process Mined: B2B Commerce (Sales Cloud), SAP

2. International Rollout — Conformance Before Replication

Marelli Aftermarket has an open $115k intl rollout opp and a $73k Geographic Extension Wave 2 opp — expanding the Italian aftermarket ops model into Poland, Germany, Iberia, France and beyond. Every country that goes live with divergent variants of order-management, warranty and returns hardens duplicate work the group will pay for indefinitely. AFPI mines Italy as the reference process and produces the conformance dashboard the rollout PM uses to force each new country onto the reference variant at go-live rather than after. Metric moved: cross-country process conformance %, time-to-productive-go-live, post-go-live rework tickets.

Key System(s) to be Process Mined: Sales Cloud, MuleSoft

3. Warranty & Returns — Claims Cycle Across Distributors

Aftermarket parts (Magneti Marelli branded starters, alternators, lighting, suspension, brakes) carry warranty exposure that flows back through the distributor network as returns and credit-notes. Claim cycles that drag hurt margin twice: reserve carry cost and distributor NPS. AFPI mines the warranty-claim event log across the Call Center, Sales Cloud case objects and SAP credit-note flow to expose why some claims sit 45+ days, which distributors are outliers, and where Apromore's AI Copilot can flag anomalies live. Metric moved: warranty cycle time, reserve accuracy, distributor claim NPS.
